What to look for first

  1. What type of plan is this?
  2. Is the provider in-network or out-of-network?
  3. Is the issue a referral problem or an authorization problem?
  4. Was coverage active on the date of service?
  5. Is the denial really a denial, or is it patient responsibility?
  6. Does the code point to invalid information, missing information, network rules, timing, or benefits?

Code bank reminders

CO-197 = precertification / authorization / notification absent
CO-242 = services not provided by network / primary care providers
CO-243 = services not authorized by network / PCP
PR-1 = deductible amount
CO-29 = timely filing limit expired
CO-16 = claim lacks information needed for adjudication
Error 55 = subscriber / insured not found
Error 74 = member ID does not match payer records
Error 45 = provider is out of network for member
A7 = rejected for invalid information
A6 = rejected for missing information

A realistic claim is messy. The first thing students notice is not always the real issue. Slow down and separate the clue from the answer.
